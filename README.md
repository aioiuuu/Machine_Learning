##  Contents Overview

### 1. 经典机器学习算法
- **回归类**：线性回归、逻辑回归
- **分类/集成类**：KNN、朴素贝叶斯、决策树、随机森林、AdaBoost、GBDT、XGBoost
- **降维/优化类**：PCA（主成分分析）、梯度下降、正则化（L1/L2/Dropout）

### 2. 深度学习入门模型
- **基础神经网络**：ANN（全连接神经网络）
- **计算机视觉**：CNN（卷积神经网络）
- **序列模型**：RNN（循环神经网络）
- **NLP 基础**：基础自然语言处理相关实验

### 3. 其他实践
- 学习率调优实验
- 张量基础操作
- Python 小项目与数据处理练习

##  Purpose of This Repository

-  巩固机器学习与深度学习的核心理论，通过代码实现加深理解
-  记录经典算法的实现细节、调参过程与实验结果
-  作为个人学习笔记与知识体系的梳理
-  为后续更复杂的深度学习项目打下基础
